# Auto-Login Project

## 项目简介
本项目用来登陆京东供应商页面并下载订单数据，主要功能包括

- 自动填写用户名和密码
- 自动进行滑块验证码校验
- 自动下载订单数据

![image](./show.gif)


## 安装指南
1. 克隆项目到本地：
    ```bash
    git clone https://github.com/yourusername/auto-login.git
    ```
2. 进入项目目录：
    ```bash
    cd auto-login
    ```
3. 安装依赖：
    ```bash
    npm install
    ```

## 使用说明
1. 配置用户信息：
    在 `config.json` 文件中填写您的用户名和密码。
2. 运行项目：
    ```bash
    npm start
    ```
